2 kleuren looplicht

Voor ons lichtsnoer hebben we gebruik gemaakt van de shelly duo rgbw ledlamp. Dit is een wifi gestuurde lamp met tal van mogelijkheden, ze kan aangestuurd worden:

  • via een app
  • met MQTT
  • vanuit de cloud
  • rechtstreeks met een API

Uiteraard is er in dit project voor de API gekozen. De belangrijkste reden waarom ik voor shelly heb gekozen is uiteindelijk niet de hardware op zich.
Ik heb vooral voor Shelly gekozen omdat hun producten erg goed gedocumenteerd zijn. Op https://shelly-api-docs.shelly.cloud/ is een massa informatie te vinden over shelly toestellen die zo goed uitgewerkt is dat ik er alle vertrouwen in heb dat ook een volslagen beginner als ik er mee aan de slag kan.

We gebruiken hier spaceapi, een API die ook door Voidwarranties gebruikt wordt, om te laten zien of Voidwarranties al dan niet open is op dit moment.
